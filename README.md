# BakeHub

Full Stack Cloud Kitchen Bakery Ordering Platform built using Spring Boot, React, MySQL and Docker.

## Features

- User authentication with JWT
- Browse bakery products by category
- Custom cake ordering
- Shopping cart and checkout
- Order tracking
- Coupon management
- Admin dashboard
- Product and order management

## Tech Stack

**Frontend**
- React
- Vite
- CSS

**Backend**
- Spring Boot
- Spring Security
- Spring Data JPA

**Database**
- MySQL

**Tools**
- Docker
- Git

## Getting Started

1. Clone the repository.
2. Configure the MySQL database.
3. Run the Spring Boot backend.
4. Start the React frontend using Vite.
